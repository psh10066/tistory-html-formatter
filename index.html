Tistory HTML 편집기

Tistory에서 복사 붙여넣기로 인해 발생하는 불필요한 스타일을 자동으로 제거하고,
본문을 깔끔하게 포맷팅해주는 편집기

GitHub
📝 사용 방법
  1. 원하는 포맷팅 옵션을 선택합니다.
  2. Tistory 에디터에서 작성 모드를 HTML로 바꾸고 전체 내용을 복사합니다.
  3. 왼쪽 입력창에 붙여넣기 하고 오른쪽 입력창에 자동으로 포맷팅된 결과를 확인합니다.
  4. '결과 복사' 버튼을 클릭하여 포맷팅된 HTML을 복사하고 Tistory 에디터에 붙여넣기 합니다.
  5. Tistory 에디터에서 작성 모드를 기본모드로 바꾸고 내용을 확인합니다.
포맷팅 옵션
macOS에서 생성되는 백스페이스 특수문자를 제거합니다
공백이 중복으로 들어간 경우 정리하고,
글의 맨 앞, 맨 뒤 공백을 제거합니다
링크의 한글 주소를 디코딩하여 잘 보이도록 합니다
색상, 배경색, 정렬, 폰트, 자간 등의 스타일을 제거합니다
